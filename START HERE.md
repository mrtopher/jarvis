# Welcome

This vault is a **sanitized, plug-and-play EMAI Obsidian OS** package. It keeps the structure, theme, plugins, and operating style — without personal notes or private vault state.

## What this edition is
This is the **free starter edition with the daily operating system core**.

## Fastest path
1. Open this folder as an Obsidian vault.
2. Let plugins finish loading.
3. Read `System/Setup Guide.md` if you need tool recommendations.
4. Run `/start` in your agent harness.
5. Then run `/interview` to personalize the vault.
6. Start using `/today`, `/new`, `/closeday`, and the rest of the included commands.

## Included commands
- `/start`
- `/interview`
- `/today`
- `/new`
- `/closeday`
- `/meeting-notes`

## Key folders
- `00 Human/` — your notes, tasks, projects, people, context
- `Machine/` — workflows, templates, skills, personalization files, outputs
- `System/` — setup and operating docs

## Important
- Open your AI harness from the **vault root** so relative command paths work.
- `/interview` regenerates the personalization files used by `/today` and `/closeday`.
- `emai-command-center` is intentionally excluded from this distribution.
